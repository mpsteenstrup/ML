# Matematikken bag Machine Learning

Projekt med 12 film om hvordan matematik bliver brugt i danske virksomheder. Denne er om hvordan machine-learning bliver brugt i 3Shape, DMI og B&O. Til filmene er der udviklet undervisningsmateriale til matematikundervisningen. 

Film og materiale kan findes her: [LRu Matematikken bag Machine Learning](https://lru.praxis.dk/Lru/microsites/virksomhederiundervisningen/ml.html)

Til at køre simuleringer er der udviklet en jupyter notebook som kan tilgås her:

* [Binder online machineLearning1.ipynb](https://mybinder.org/v2/gh/mpsteenstrup/ML/main?labpath=traekVirksomhederneIndIUndervisningen%2FmachineLearning1.ipynb)
* [Fil til download machineLearning1.ipynb](https://github.com/mpsteenstrup/ML/blob/main/traekVirksomhederneIndIUndervisningen/machineLearning1.ipynb)




Der er to tilhørende filer, [teeth.png](teeth.png) og [AirPassengers.csv](AirPassengers.csv), som skal uploades til jupyter filen.
